项目功能： 全球关键矿产宏观政策情报雷达 —— 自动化监控多国政府对锂、钴、镍、铜、稀土、石墨六类关键矿产的出口禁令、资源民族主义、关税调整等政策变动，依赖 DeepSeek v4-pro 大模型进行结构化战略分析，并将研判结果持久化至 Notion 知识库、高冲击力政策通过钉钉推送给决策层。

技术栈： Python 3.10+ / OpenAI SDK (DeepSeek v4-pro) / BeautifulSoup4 + lxml (HTML/RSS解析) / PyYAML (配置) / requests (HTTP) / Notion API + DingTalk Webhook / GitHub Actions (定时调度)
